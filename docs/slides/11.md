# Practical example 3: Test fixutres

Let's add a fixture:

```python tests/test_using_pytest.py
import pytest
import using_pytest

@pytest.fixture
def data():
    return [3, 4]

def test_add_data(data):
    assert using_pytest.add(*data) == 7

def test_multiply_data(data):
    assert using_pytest.multiply(*data) == 12
```
